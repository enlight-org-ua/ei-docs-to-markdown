# Enlight Docs to Markdown

**Enlight Docs to Markdown** — це безкоштовний інструмент, який допомагає швидко перетворювати PDF, Word, PowerPoint, HTML та зображення у Markdown — чистий, структурований текстовий формат, зручний для подальшої роботи з AI/ШІ.

Відкрити інструмент у Google Colab:  
https://colab.research.google.com/github/enlight-org-ua/ei-docs-to-markdown/blob/main/Enlight_Docs_to_Markdown.ipynb

## Для чого це потрібно

Ми часто даємо штучному інтелекту різні файли: PDF-документи, Word-файли, презентації, скани, зображення, інструкції, звіти або великі пакети матеріалів — і просимо щось упорядкувати, проаналізувати, перекласти, скоротити, структурувати або перетворити в новий документ.

Але важливо пам’ятати: ШІ не “бачить” документ так, як людина. Він працює з текстом через токени — невеликі фрагменти інформації, з яких модель складає розуміння документа. Якщо ми завантажуємо у ШІ файл із великою кількістю зайвого шуму, модель витрачає частину своєї уваги не на зміст, а на сміття.

У Word, PDF або презентаціях може бути дуже багато зайвого: повторювані колонтитули, номери сторінок, службовий текст, розриви рядків, непотрібне форматування, приховані елементи, таблиці без структури, декоративні блоки, дублікати, візуальні елементи, які не несуть змісту. Іноді до 90% того, що передається в AI, не є корисними даними для аналізу.

У результаті штучний інтелект може:

- витрачати токени на зайвий текст;
- гірше розуміти структуру документа;
- пропускати важливі фрагменти;
- давати менш точні відповіді;
- плутати основний зміст із технічним або візуальним шумом;
- швидше досягати ліміту контексту;
- аналізувати не весь документ, а лише частину того, що помістилося в контекст.

## Що робить цей інструмент

Цей notebook створений, щоб будь-хто міг безкоштовно підготувати свої документи до якіснішої роботи з AI/ШІ.

Інструмент:

- приймає PDF, Word, PowerPoint, HTML та зображення;
- перетворює документи у Markdown;
- прибирає частину зайвого візуального та технічного шуму;
- робить текст більш структурованим;
- допомагає зменшити витрати токенів;
- покращує якість подальшого аналізу ШІ;
- дозволяє швидше готувати звіти, інструкції, бази знань, резюме, переклади, контент для сайтів або матеріали для подальшої обробки.


## Чому Markdown краще для роботи з AI

Markdown — це простий текстовий формат із базовою структурою: заголовками, списками, таблицями, посиланнями та блоками тексту. Для AI такий формат зазвичай зрозуміліший, ніж складний PDF або презентація.

Коли ШІ отримує Markdown замість “важкого” документа, він краще бачить логіку матеріалу:

- де заголовок;
- де основний текст;
- де список;
- де таблиця;
- де окремі розділи;
- де починається і закінчується смисловий блок.

Це допомагає моделі краще аналізувати документ і давати більш точні, повні та корисні відповіді.

## Як це допомагає зменшити використання токенів

Кожен зайвий фрагмент тексту в документі — це додаткові токени. Якщо у файл потрапляють повторювані колонтитули, службові написи, хаотичне форматування або дублікати, ШІ все одно намагається це обробити.

Конвертація в Markdown допомагає підготувати чистішу версію документа. Це не гарантує ідеального очищення в усіх випадках, але часто суттєво зменшує кількість зайвого тексту та робить структуру зрозумілішою.

Менше сміття в документі означає:

- менше зайвих токенів;
- більше місця для реального змісту;
- кращу якість аналізу;
- менше ризику, що модель пропустить важливу інформацію;
- зручнішу роботу з великими документами.
